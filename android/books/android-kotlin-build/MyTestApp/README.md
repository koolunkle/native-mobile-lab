# MyTestApp

## Overview
A project dedicated to demonstrating unit testing strategies and framework integration in Android.

## Key Learning Topics
- Unit Testing with JUnit and Mockito
- Using Robolectric for Android-dependent unit tests
- Implementing custom Test Rules and handling error cases

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** JUnit, Mockito, Robolectric, Espresso, AndroidX JUnit KTX

## Notes
Focuses on writing maintainable and robust unit tests, covering both pure Kotlin logic and Android-specific components without requiring a physical device.
