## Overview
A demonstration of Android Activity launch modes and their impact on the Back Stack and Task management.

## Key Learning Topics
- Activity launch modes: standard and singleTop
- Task stack behavior and navigation between multiple activities
- Intent-based activity transitions
- Manifest-level activity configuration

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML
- **Key Libraries:** AndroidX Core KTX, AppCompat, Material Components

## Notes
The project uses multiple activities (ActivityA, ActivityB, ActivityC, ActivityOne, ActivityTwo, ActivityThree) to illustrate how different launch mode settings affect the instantiation and ordering of activities within a task.
