## Overview
An implementation of the Bottom Navigation pattern using the Jetpack Navigation Component to manage application destinations.

## Key Learning Topics
- Integrating BottomNavigationView with NavController
- Defining application destinations in a Navigation Graph (XML)
- Managing Fragment transactions automatically through NavHostFragment
- Synchronizing UI state with the current navigation destination

## Tech Stack & Libraries
- **Architecture:** Basic (Navigation Component)
- **UI:** XML with ViewBinding
- **Key Libraries:** Jetpack Navigation Component, Material Components, AndroidX Fragment KTX

## Notes
The application demonstrates how to set up a standard bottom navigation layout where each menu item corresponds to a specific Fragment destination, with the Navigation Component handling the back stack and UI updates.
