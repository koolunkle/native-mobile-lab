## Overview
A location-based application demonstrating Google Maps integration and the Android runtime permission system.

## Key Learning Topics
- Integrating Google Maps SDK for Android
- Requesting and handling runtime location permissions (ACCESS_FINE_LOCATION)
- Using FusedLocationProviderClient to retrieve the device's current location
- Adding markers and customizing the Map UI

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** Google Play Services Maps, Google Play Services Location, Material Components

## Notes
The project illustrates the complete workflow for a maps-based app, including API key configuration (via Secrets Gradle Plugin), permission checks, and interactive map markers.
