# FragmentIntro

## Overview
An introductory project that demonstrates the basic concepts of Android Fragments, including their definition and static inclusion in activity layouts.

## Key Learning Topics
- Defining Fragment classes and layouts
- Static fragment inclusion in XML
- Basic relationship between Activities and Fragments

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX AppCompat, Material Components

## Notes
This project serves as a foundational example for understanding how fragments can be used to modularize the user interface.
