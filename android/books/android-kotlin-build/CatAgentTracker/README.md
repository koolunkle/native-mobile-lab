# CatAgentTracker

## Overview
A project demonstrating background processing and route tracking using Android Services and WorkManager.

## Key Learning Topics
- Foreground and Background Services
- Scheduled tasks with WorkManager
- System Notifications
- Permissions handling for location and background tasks

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** WorkManager, AndroidX Core

## Notes
The project explores the trade-offs between using a long-running Foreground Service for real-time tracking versus WorkManager for periodic background operations.
