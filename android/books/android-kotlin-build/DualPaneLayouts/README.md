# DualPaneLayouts

## Overview
A project demonstrating how to implement a dual-pane layout for different screen sizes, such as phones and tablets, using fragments to create an adaptive user interface.

## Key Learning Topics
- Master-Detail pattern implementation
- Adaptive layout design for various screen configurations
- Fragment-to-Activity communication and coordination

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX AppCompat, Material Components

## Notes
The project shows how to handle single-pane (phone) and dual-pane (tablet) configurations by checking for the presence of a second container in the activity layout.
