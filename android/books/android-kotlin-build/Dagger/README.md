# Dagger

## Overview
A project demonstrating the implementation of automated Dependency Injection using the Dagger 2 library in an Android environment.

## Key Learning Topics
- Dagger Components, Modules, and Providers
- Scopes and Subcomponents management
- Constructor and Field Injection
- Integration with ViewModel using custom factories

## Tech Stack & Libraries
- **Architecture:** MVVM
- **UI:** XML with ViewBinding
- **Key Libraries:** Dagger 2, ViewModel, LiveData

## Notes
Showcases a hierarchical dependency graph with an ApplicationComponent and feature-specific Subcomponents to manage object lifecycles effectively.
