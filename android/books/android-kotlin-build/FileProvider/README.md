# FileProvider

## Overview
A project focused on secure file sharing between applications using the FileProvider component and content URIs instead of file URIs.

## Key Learning Topics
- Secure file sharing with Content URIs
- FileProvider configuration and path mapping in AndroidManifest.xml
- Handling file permissions for external applications

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX Core, Apache Commons IO

## Notes
The project illustrates best practices for sharing files securely, ensuring that sensitive data is only accessible via temporary, granted permissions.
