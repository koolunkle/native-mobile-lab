# MediaStore

## Overview
A project demonstrating how to interact with the Android MediaStore API to query, read, and write media files while respecting modern storage restrictions.

## Key Learning Topics
- Using ContentResolver to query media
- Handling Scoped Storage requirements
- Managing file permissions for external storage
- Performing file operations with MediaStore URIs

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX Core, Commons IO, Material Components

## Notes
Includes utility classes like FileHelper and MediaContentHelper to abstract the complexities of working with ContentProviders and different Android API levels.
