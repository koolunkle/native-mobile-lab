# IntentsIntroduction

## Overview
A fundamental project demonstrating the use of Intents for navigating between different Activities and passing data within an Android application.

## Key Learning Topics
- Explicit and Implicit Intents
- Starting activities for results
- Data passing using Intent Extras
- Activity lifecycle during navigation

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX AppCompat, Material Components

## Notes
Focuses on the core communication mechanism between Android components, illustrating how to transition from MainActivity to WelcomeActivity with user data.
