# PopularMovies

## Overview
A comprehensive movie discovery application that fetches popular movies from a remote API and implements an offline-first strategy with local caching.

## Key Learning Topics
- Clean Architecture implementation (Domain, Data, and UI layers)
- Dependency Injection with Hilt
- Paging 3 with RemoteMediator for efficient list loading and caching
- Offline-first approach using Room database
- Reactive programming with Kotlin Flow and Coroutines

## Tech Stack & Libraries
- **Architecture:** MVVM with Clean Architecture
- **UI:** XML with ViewBinding and DataBinding
- **Key Libraries:** Hilt, Room, Retrofit, Paging 3, Glide, WorkManager

## Notes
This project demonstrates modern Android development standards, including the Repository pattern, Use Cases, and robust state management. It also utilizes WorkManager for background tasks and Retrofit for network communication.
