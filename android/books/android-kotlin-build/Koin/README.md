# Koin

## Overview
A project demonstrating the use of Koin, a pragmatic and lightweight dependency injection framework for Kotlin developers, to manage dependencies in an Android app.

## Key Learning Topics
- Koin DSL (module, single, factory)
- ViewModel injection with Koin
- Application level Koin initialization
- Dependency injection without code generation

## Tech Stack & Libraries
- **Architecture:** MVVM
- **UI:** XML with ViewBinding
- **Key Libraries:** Koin, LiveData, ViewModel

## Notes
Provides an alternative to Hilt/Dagger, showing how to achieve clean architecture by decoupling component creation from usage through Koin modules.
