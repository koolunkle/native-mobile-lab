# FragmentLifecycle

## Overview
A project designed to explore and demonstrate the various stages of the Fragment lifecycle through logging and visual state indicators.

## Key Learning Topics
- Fragment lifecycle callback methods (onAttach through onDetach)
- Understanding the relationship between Fragment and Activity lifecycles
- Best practices for state management within lifecycle events

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX AppCompat, Material Components

## Notes
The project provides insights into when specific lifecycle methods are triggered, which is crucial for managing resources and UI updates correctly.
