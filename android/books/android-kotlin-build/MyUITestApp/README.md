# MyUITestApp

## Overview
An application focused on instrumentation tests and UI automation using the Espresso framework.

## Key Learning Topics
- UI Testing with Espresso and Test Rules
- Custom Test Runners and Application classes for testing
- Synchronization using Idling Resources

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** Espresso, Test Rules, Idling Resource, Navigation Component, AppCompat

## Notes
Demonstrates advanced UI testing techniques, including testing navigation flows and handling asynchronous operations through Idling Resources.
