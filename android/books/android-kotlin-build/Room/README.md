# Room

## Overview
This project demonstrates how to use the Room persistence library to manage a local SQLite database, including complex entities and relationships.

## Key Learning Topics
- Room entities, DAOs, and Database configuration
- Type converters for custom data types
- Entity relationships (one-to-one, one-to-many, many-to-many)
- RxJava integration for reactive database operations

## Tech Stack & Libraries
- **Architecture:** Basic (Room persistence layer)
- **UI:** XML with ViewBinding
- **Key Libraries:** Room (Runtime, KSP, RxJava3), RxJava

## Notes
Features advanced Room usage such as joined entities and type converters for handling non-primitive data types like Date.
