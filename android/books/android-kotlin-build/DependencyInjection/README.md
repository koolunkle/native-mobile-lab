# DependencyInjection

## Overview
A project demonstrating the foundational principles of Dependency Injection using the manual Container pattern.

## Key Learning Topics
- Manual Dependency Injection principles
- Application-level and Activity-level Dependency Containers
- Decoupling object creation from business logic
- Implementing custom ViewModelProvider.Factory for dependency injection into ViewModels

## Tech Stack & Libraries
- **Architecture:** MVVM
- **UI:** XML with ViewBinding
- **Key Libraries:** ViewModel, LiveData

## Notes
This project illustrates how Dependency Injection works under the hood without automated frameworks, providing a clear understanding of object graph management and lifecycle-aware dependencies.
