# MyComposeApp

## Overview
A modern Android application demonstrating Jetpack Compose for UI development, including state management and navigation.

## Key Learning Topics
- Declarative UI development with Composable functions
- State management and hoisting in Compose
- Navigation between screens using Navigation Compose

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** Jetpack Compose (Material 3)
- **Key Libraries:** Navigation Compose, Activity Compose, Compose BOM, Lifecycle Runtime KTX

## Notes
Focuses on the transition from imperative XML-based UI to declarative Compose-based UI, showcasing modern Android development practices.
