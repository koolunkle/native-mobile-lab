# TabNavigation

## Overview
A project demonstrating how to implement tabbed navigation using TabLayout and ViewPager2 to switch between different fragments.

## Key Learning Topics
- TabLayout integration and configuration
- ViewPager2 for horizontal paging
- FragmentStateAdapter implementation
- Synchronization between TabLayout and ViewPager2

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** Material Components, ViewPager2

## Notes
The project showcases the use of scrollable tabs (app:tabMode="scrollable") to accommodate multiple categories and how to link them seamlessly with a ViewPager2 component.
