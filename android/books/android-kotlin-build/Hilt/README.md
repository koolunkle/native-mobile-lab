# Hilt

## Overview
A project demonstrating dependency injection in Android using the Hilt library to simplify DI implementation and improve code maintainability.

## Key Learning Topics
- Hilt setup and configuration
- Field and constructor injection
- ViewModel injection with @HiltViewModel
- Dependency scoping and modules

## Tech Stack & Libraries
- **Architecture:** MVVM
- **UI:** XML with ViewBinding
- **Key Libraries:** Hilt, AndroidX Activity KTX, AndroidX Fragment KTX

## Notes
Implements a repository pattern with NumberRepository and NumberRepositoryImpl, showcasing how Hilt handles interface binding and dependency management.
