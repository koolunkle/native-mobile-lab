# DataStore

## Overview
A project demonstrating modern data persistence in Android using Jetpack DataStore (Preferences) as a replacement for SharedPreferences.

## Key Learning Topics
- Jetpack DataStore (Preferences) implementation
- Reactive data streams with Kotlin Flows
- Asynchronous data operations with Coroutines
- Data persistence lifecycle management

## Tech Stack & Libraries
- **Architecture:** MVVM
- **UI:** XML with ViewBinding
- **Key Libraries:** Preferences DataStore, ViewModel, LiveData, Coroutines

## Notes
Focuses on the benefits of DataStore over SharedPreferences, including its thread-safety, error handling, and support for reactive programming.
