# NavigationDrawer

## Overview
This project demonstrates the implementation of a Navigation Drawer using the Jetpack Navigation component and Material Design components to provide a standard side navigation menu.

## Key Learning Topics
- DrawerLayout and NavigationView implementation
- Jetpack Navigation component integration for side navigation
- AppBarConfiguration for managing top-level destinations

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding, Material Design Components
- **Key Libraries:** Navigation Component (Fragment, UI KTX), Material Components

## Notes
Uses AppBarConfiguration to automatically handle the drawer icon and back button behavior in the Action Bar based on the current destination.
