# TipCalculator

## Overview
A utility application that calculates tip amounts based on user input, focusing on basic UI components and activity-to-activity data transfer.

## Key Learning Topics
- User input handling with EditText
- Data passing between activities via Intent
- Activity lifecycle and layout management
- Basic mathematical logic in Kotlin

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** Material Components

## Notes
The application consists of a primary input screen and a secondary output activity to display calculated results, illustrating the fundamental concept of navigation and state sharing between components.
