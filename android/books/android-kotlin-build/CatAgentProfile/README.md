# CatAgentProfile

## Overview
A project demonstrating how to fetch and display data from a remote API using Retrofit and image loading with Glide.

## Key Learning Topics
- Networking with Retrofit
- JSON parsing with Moshi and KSP (Kotlin Symbol Processing)
- Image loading and caching with Glide
- View Binding for type-safe layout access

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** Retrofit, Moshi, Glide, KSP

## Notes
The project uses a custom ImageLoader interface to abstract the image loading implementation, allowing for easier testing and swapping of libraries.
