# DynamicFragments

## Overview
This project demonstrates how to dynamically manage fragments at runtime, including adding, removing, and replacing them within an activity.

## Key Learning Topics
- FragmentManager and FragmentTransaction usage
- Dynamic UI manipulation without restarting the activity
- Managing the fragment backstack

## Tech Stack & Libraries
- **Architecture:** Basic
- **UI:** XML with ViewBinding
- **Key Libraries:** AndroidX Fragment KTX, Material Components

## Notes
The implementation focuses on the programmatic control of fragments, illustrating how to use the FragmentManager to update the UI based on user interaction.
