# JetpackFragments

## Overview
A project demonstrating the modern approach to fragment management using Android Jetpack's Navigation component for a structured UI flow.

## Key Learning Topics
- Navigation Graph implementation
- NavHostFragment and NavController
- Fragment transactions and backstack management
- Safe Args for type-safe data passing

## Tech Stack & Libraries
- **Architecture:** Basic with Navigation Component
- **UI:** XML with ViewBinding
- **Key Libraries:** Navigation Component, Material Components

## Notes
Demonstrates a master-detail flow using ListFragment and DetailFragment, showcasing how the Navigation component simplifies complex UI transitions.
